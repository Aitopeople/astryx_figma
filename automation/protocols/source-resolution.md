# Source resolution

1. Resolve the exact installed core/CLI version and reuse its hash-bound official contract cache.
2. CLI component/template output is the source of truth for public API and example code.
3. Use package source only when the cached contract lacks geometry, state, or runtime anatomy.
4. Use rendered website/MCP only for current visible coverage, guidance, or a recorded source conflict.
5. Recollect globally only when package, CLI, lockfile, site/crawl version, or source hash changes.
6. Never infer official content from existing Figma art.

Escalate conflicts as blocked differences; do not choose a source silently.
